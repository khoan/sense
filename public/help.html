Elasticsearch Shell Help

Usage

  • Enter URL of Elasticsearch server
  • Enter Query, then press Ctrl+Enter to obtain result

URL

URL input supports format as outlined in RFC3986.

Query

Query box reduces RSI with tab autocomplete.

You can have multiple queries in query box, but only the one under cursor will be submitted when Ctrl+Enter is pressed.

Format

Query format is:

<verb> <path>
<optional body>

example

GET :index/:type/_search
{
  "query": {
    "match_all": {}
  }
}
